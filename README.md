2D RPG Game
A 2D tile-based RPG built in Java using Swing/AWT with a custom game engine built from scratch.
Features

60 FPS game loop using a dedicated game thread with delta-time rendering
Tile-based world with a 50x50 map and scrolling camera
WASD movement with directional collision detection against tiles and objects
NPC system with a merchant character
Interactable objects — keys, doors, chests, and boots
Sound system — looping background music and sound effects
Pause menu with keyboard toggle (P)
Debug mode — press T to display real-time draw time overlay

Controls
Key 
Action
W / A / S / D Move
P Pause / Unpause
T Toggle debug draw time
How to Run

Clone the repository
Open in any Java IDE (IntelliJ, VS Code, Eclipse)
Run Main.java

Built With
